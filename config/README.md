# Configuration Files

This directory contains configuration files for the IRH project.

## Files

- **config.json** - General project configuration
- **project_config.json** - Project-specific configuration

## Usage

These configuration files are used by various scripts and tools in the repository. They are organized here to keep the root directory clean.

## Related Files

- Environment configuration: `/environment.yml`
- Python project configuration: `/pyproject.toml`
- Installation configuration: `/setup.py`
